# conda指令 
### conda版本
```shell
conda --version
```
### 創造環境
```shell
conda create -n envname python=3.8
```
### activate環境
```shell
conda activate envname
```
### deactivate環境
```shell
conda deactivate envname
```
### remove環境
```shell
conda remove --name envname --all
```
### install packages
```shell
conda install numpy matplotlib
```
### Launch the Jupyter Lab
```shell
jupyter lab
```
